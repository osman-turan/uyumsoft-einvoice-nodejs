# uyumsoft-einvoice-nodejs

Minimal Node.js app for Uyumsoft E-Invoice Integration.

This is a minimal Node.js app that connects to Uyumsoft E-Invoice Integration SOAP Service. The app is configured to use the test environment of the service.

## Getting Started

After cloning the project, install Node.js dependencies with the following:

```bash
$ node install
```

Then, you can run the script as the following:

```bash
$ node index.js
```
